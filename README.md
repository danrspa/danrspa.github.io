# ᚺᛖᚱ ᛒᛁᚱ ᚢᚯᛚᚢᚨᚾ

### *Hér býr vǫlvan.*
### *Gakk inn — ok verðr þín framtíð at fortíð þinni.*

᛫ ᚠ ᚢ ᚦ ᚨ ᚱ ᚲ ᚺ ᚾ ᛁ ᛃ ᛖ ᛈ ᛉ ᛋ ᛏ ᛒ ᛗ ᛚ ᛜ ᛞ ᛟ ᛫

Í hverri viku rís ný spá ór djúpinu;
hin gamla sígr niðr í **salinn langa**.

Heimrinn andar — borgir rísa ok hníga, mánann vex ok þverr,
vetrinn kømr ok ferr — ok vǫlvan heyrir þat allt.
En hon nefnir ekki; hon kveðr einungis.

Spyr eigi hvaðan spáin kømr.
Lestu hana, ok lát hana fara.

> *Eldrinn sem þú geymir er hin eina lǫg sem þú átt.*
